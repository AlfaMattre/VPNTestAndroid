## Как запустить проект

### Вариант 1. Через Android Studio
1. Открыть проект в Android Studio
2. Дождаться завершения Gradle Sync
3. Убедиться, что используется JDK 17
4. Запустить приложение на эмуляторе или физическом устройстве

### Вариант 2. Через APK
Приложение также можно запустить с помощью готового APK, размещенного в Git-репозитории.

## UI улучшения

- был полностью переработан интерфейс в стиле современных VPN-приложений
- была добавлена анимация для кнопки подключения
- была подобрана цветовая схема, визуально близкая к современным VPN-приложениям
- при активном VPN вокруг кнопки подключения отображается pulsing-анимация в виде волн
- была улучшена визуальная иерархия главного экрана

## UX улучшения

- список стран был перенесен в Bottom Sheet, чтобы главный экран оставался чище
- популярные страны были вынесены в отдельную секцию для более быстрого выбора
- ошибки загрузки списка стран теперь отображаются внутри Bottom Sheet
- выбор страны стал более очевидным и удобным
- при смене страны активное VPN-подключение автоматически отключается, чтобы избежать некорретного состояния
- при отсутствии интернета пользователь получает ошибку подключения
- при потере интернета во время активной сессии VPN автоматически отключается
